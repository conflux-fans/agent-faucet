Agent Faucet Skill

Bootstrap chain funds without a human faucet.

A local proof-of-work flow for agents that need native gas token or a known ERC20 test token.

Install the skill 
$ bun skill/scripts/read-config.ts
enabled: true
target: 0x0000...

$ bun skill/scripts/compute-proof.ts --confirm-compute
nonce: 0x6f2a
digest: 0x0000c9...

$ bun skill/scripts/submit-claim.ts --proof proof.json
txHash: 0x8bcc...

What it does

Three scripts, one claim path.

  1. 01 Read chain config

    Checks the static deployment index, token status, target, and cooldown through RPC.

  2. 02 Compute proof locally

    Runs the only CPU-heavy step after explicit user confirmation.

  3. 03 Submit through serverless

    Posts the proof to a stateless relayer that simulates and returns a transaction hash.

Install

Add the skill, then run the scripts.

bun skill/scripts/read-config.ts --chain-id 31337 --rpc-url http://127.0.0.1:8545 --recipient 0x... --token native
bun skill/scripts/compute-proof.ts --confirm-compute --chain-id 31337 --rpc-url http://127.0.0.1:8545 --recipient 0x... --token native
bun skill/scripts/submit-claim.ts --proof proof.json
