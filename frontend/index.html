Agent Faucet Skill

Claim Test Chain Funds from an Agent Workflow 让 Agent 在工作流中领取测试链资金

Agent Faucet is a repo-backed skill that lets agents request native gas token or a known ERC20 test token without using a browser faucet, wallet UI, CAPTCHA, or social login. Agent Faucet 是一个可从仓库安装的 Skill，让 Agent 可以脚本化领取 native gas token 或已知 ERC20 测试币， 不需要浏览器 faucet、钱包 UI、验证码或社交登录。

Send the Skill to your agent 把 Skill 文件发给 Agent

https://github.com/conflux-fans/agent-faucet/blob/main/skills/agent-faucet/SKILL.md

Use Vercel Skills CLI 用 Vercel Skills CLI

npx skills add https://github.com/conflux-fans/agent-faucet --skill agent-faucet

Availability 可用部署

Supported Chain and Claimable Token 当前支持的链与可领取 Token

Network 网络 Conflux eSpace Testnet

Claimable Token 可领取 Token Native CFX 原生 CFX
