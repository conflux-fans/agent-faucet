Agent Faucet Skill

Claim Test Chain Funds from an Agent Workflow 让 Agent 在工作流中领取测试链资金

Agent Faucet is a repo-backed skill that lets agents request native gas token or a known ERC20 test token without using a browser faucet, wallet UI, CAPTCHA, or social login. Agent Faucet 是一个可从仓库安装的 Skill，让 Agent 可以脚本化领取 native gas token 或已知 ERC20 测试币， 不需要浏览器 faucet、钱包 UI、验证码或社交登录。

Install 安装

Two Ways to Install 两种安装方式

Send the Skill to your agent 把 Skill 文件发给 Agent

Share the repository's skills/agent-faucet folder with your agent. It contains SKILL.md, scripts, references, and the deployment index. 把仓库里的 skills/agent-faucet 文件夹交给你的 Agent。里面包含 SKILL.md、脚本、参考文档和部署索引。 

https://github.com/conflux-fans/agent-faucet/tree/main/skills/agent-faucet

Install with Vercel Skills CLI 用 Vercel Skills CLI 安装

Use the npx skills installer when your agent supports the open agent skills format. 如果你的 Agent 支持开放的 agent skills 格式，可以用 npx skills 安装器直接安装。 

npx skills add https://github.com/conflux-fans/agent-faucet --skill agent-faucet

Runtime requirements: Bun and Foundry cast. Setup details live in skills/agent-faucet/reference/install-requirements.md. 运行要求：Bun 和 Foundry cast。安装细节见 skills/agent-faucet/reference/install-requirements.md。
